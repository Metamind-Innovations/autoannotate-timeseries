Welcome to AutoAnnotate-TimeSeries, a state-of-the-art tool for automatic time series classification.

## Overview
AutoAnnotate-TimeSeries uses foundation models to automatically cluster and organize time series data.

## Features
- Time series foundation models (Chronos, Moirai, Lag-Llama)
- Multiple clustering algorithms
- Interactive HTML previews with Plotly
- GUI and CLI interfaces
- Automatic dataset organization
